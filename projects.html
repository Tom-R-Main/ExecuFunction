Demos

Small, honest proofs of the system—each returns JSON you can curl. These are building blocks for WinonaOS.

Calendar orchestration (next-3)

Returns the next three events. Works with a demo ICS feed today; tokenized calendar later.

curl https://www.execufunction.com/api/calendar/next3

Ritual logging (2-minute check-in)

A tiny endpoint that records a short check-in. No-store on transit; minimal fields at rest.

curl -X POST https://www.execufunction.com/api/rituals/log \
  -H "Content-Type: application/json" \
  -d '{"mood":"ok","note":"wrapped standup"}'

Tasks: ready list

Minimal task entity with an optional "next action" and predictable filters (ready/waiting/all).

Context clipboard (under the hood)

Technical demo: Returns a compact JSON envelope with calendar events and system state. Used internally by WinonaOS.
